# 🖊️ 墨痕 (MoHen) | Handwriting Lab v1.2

> **AI 笔迹克隆实验**：精准捕捉指尖的压力与神韵，为冰冷的像素注入手写的温度。

## 🔬 核心协议 (The Protocol)

项目通过三阶段闭环，确保每一枚字符都具备真实书写的质感：

1. **Prepare (模型标定)**: 下载专属 `Vision Transformer` 空间网格模板，建立高精度的书写基准线映射。
2. **Refine (样本精炼)**: 神经网络自动分割 30 个核心特征字符。用户可针对每个字符的提取边界进行手动校验，确保“炼丹”素材的纯度。
3. **Training (神经合成)**: 经过 1500 步的深度拟合（学习率 $1e-6$），构建专属的神经网络笔迹模型。

## 🎨 生成引擎 (Synthesis Engine)

在 **Generate** 模块中，我们提供了实时参数调节功能：

- **Dynamic Variation (动态变化率)**: 模拟真实书写中的随机性，避免字迹过于规整。
- **Ink Density (墨水密度)**: 调节笔触的干枯与浓重质感，还原纸墨交融的视觉体验。

## 🚀 启动与运行

- **一键开启**: 双击运行 `start.bat`。
- **一键退出**: 任务完成后，运行 `stop.bat` 释放 GPU 显存占用。
